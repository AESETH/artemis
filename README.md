## Artemis

MateCloud 前端框架,采用 vue-admin-beautiful 构建

## 前端模板

前端模板基于：vue-admin-template 生成
前端模板： [https://github.com/chuzhixin/vue-admin-beautiful](https://github.com/chuzhixin/vue-admin-beautiful)

## 安装教程

```bash
# 安装依赖
npm install
# 或者
yarn install

# 启动服务 localhost:9528
npm run serve
## <2.2.8
npm run dev
# 或者
yarn serve
## <2.2.8
yarn dev


# 构建生产环境
npm run build
```

## 环境要求

- node > 14.5
